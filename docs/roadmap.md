# Roadmap — LocalBiz SEO Auditor

## Sprint 1 — UI e estados (concluído)
- Layout semântico e responsivo
- Formulário com validação
- Estados: idle/loading/error/success
- Componentes base e acessibilidade

## Sprint 2 — Extração on-page (SEO)
- Title e meta description
- Contagem de H1/H2
- Imagens sem alt
- Links sem texto/contexto
- Checklist inicial + score

## Sprint 3 — Performance
- Integração com API de performance
- Tratamento de falhas e fallback

## Sprint 4 — Exportação e entrega
- Export JSON
- Export HTML printável
- Prints, revisão de docs e deploy (GitHub Pages)
